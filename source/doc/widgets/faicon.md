### Short declaration

`photonui.FAIcon` can be instantiated in a shorter way:

```javascript
var icon = new photonui.FAIcon("fa-camera");
photonui.domInsert(icon, "demo");
```

With more params:

```javascript
var icon = new photonui.FAIcon("fa-camera", {
    size: "fa-3x",
    color: "#DB624F"
});

photonui.domInsert(icon, "demo");
```
